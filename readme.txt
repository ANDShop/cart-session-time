== Description ==

Cart Session Time is a plugin that allows you to change the retention time of your WooCommerce cart information.

== Installation ==

1. Cart Session Time can be installed directly through your WordPress
Plugins dashboard.
1. Click "Add New" and Search for "Cart Checkout Confirm"
1. Install and Activate
